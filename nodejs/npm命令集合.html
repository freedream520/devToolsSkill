可以查看npm官网的api列表

具体细节可以参考阮一峰的标准参考教程
  • npm update module_name
  • npm install webpack@1.12.x 安装指定版本
  • npm ls -g
  • npm ls -depth 0 :依赖模块的深度展示
  • npm info module_name
  • npm search (使用cnpm search可以在浏览器中显示搜索结果，很人性化)
  • npm run (不加参数会列出所有的可执行命令，很有用哦)
  • npm run comandName (执行package.json文件中的scripts指令。会临时把本地模块添加到全局环境中，不用全局安装)
  • npm run build "build": "npm run build-js && npm run build-css" :先执行前者再执行后者
  • npm run build "build": "npm run build-js & npm run build-css" :两者并行执行
  • npm run test -- aa.js 如果要传递参数 需要在参数前面加两个-
