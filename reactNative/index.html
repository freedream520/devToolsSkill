window上流程

  • 1.安装javaJDK

    需要将jsk/bin目录配置到path环境变量中

  • 2.安装android studio

    a:进入SDKManager(可通过Android Studio菜单Tools-Android-SDK Manager)或者在sdk目录下 直接运行sdk mnager进行操作 更新sdk-23相关 设置代理地址，避免更新太慢

    通过android studio查看sdk的安装目录，将Android\Sdk\platform-tools配置到path环境变量中， 同时增加ANDROID_HOME的环境变量为：Android\Sdk（sdk的整个目录）

  • 3.安装C++环境

    cygwin或mingw等其他C++环境。编译node.js的C++模块时需要用到

  • 4.安装nodejs

  • 5.安装react-native命令行工具

    在gitrhub上下载react-native加压到本地 ，找到 react-native-cli目录 npm install -g

  • 6.创建项目

    react-native init MyProject

  • 7.启动服务

    cd MyProject react-native start

    验证服务是否启动成功 localhost:8081/index.android.bundle?platform=android

  • 8.打包APP

    cd MyProject react-native run-android

  • 9.运行app

    可以使用安卓模拟器 bluestacks ：需要设置模拟器的dev地址 ip:8081的格式（抖动模拟器，进行设置）

mac上流程

  • 必须的软件 Homebrew node react－native－cli xcode
  • 1.安装 Homebrew
  • 2.brew install nvm 安装nvm同时配置nvm环境变量
  • 3.nvm install stable 安装node同时配置node环境变量
  • 4.nrm切换淘宝源镜像
  • 5.npm install react-native-cli -g
  • 6.react-native init testProject
  • 7.react-native run-ios 打包编译文件
  • 8.使用xcode打开目录下的.xcodeproj文件 并修改RCTWebSocketExecutor.m文件的localhost为 ip：port
